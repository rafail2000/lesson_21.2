#Lesson_21.2

В данном проекте реализована вёрстка четырёх веб страниц:
каталог, категории, контакты и главная страница.
В файле main.py реализована имитация работы локального сервера,
который возвращает страницу с контактами.

Для запуска этой программы нужно набрать команду poetry init и 
запустить файл main.py.
